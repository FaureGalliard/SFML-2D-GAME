## Mecanicas:
Mecanicas principales: combate contra esqueletos, debes plantar arboles, apagar el fuego, tieninventario, hay tipos de plantas que puedes plantar, el jugador puede personalizar su personaje(solo cambiar el tipo de pelo)
interaccion: el personaje no puede atravesar objetos, pero si enemigos y plantas que todavia no han crecido del todo
Acciones del jugador: puede plantar arboles, vegetales, puede usar herramientas, tanto para matar o para 
interactuar con el entorno, puede regar plantas para que crezcan despues de un tiempo
Tipo de controles: El jugador se puede mover con el teclado o con un mando de consola
Progresion: El jugador podra guardar su partida  y su mapa
Sistema de inventario: Si, puede ir al pozo y llenar su cubeta de agua, al matar enemigos obtendra diferentes tipos de semillas.
Enemigos:  esqueletos, tendran diferentes formas de atacar
Mecanicas de tiempo: algunos esqueletos podran prender fuego, este fuego se exparsira a medida que pasa el tiempo, quemando arboles. Despues de regar una planta, esta crecera
Aleatoriedad: Los enemigos tendran diferentes acciones que seran al azar
Sistema de combate: el personaje principal podra usar diferentes acciones para matar enemigos
